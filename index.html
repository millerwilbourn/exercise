One-time setup guide
1
Go to console.cloud.google.com and create a new project (name it anything).
2
Search "Google Drive API" and enable it for your project.
3
Go to APIs & Services → Credentials → Create Credentials → OAuth Client ID. Choose "Web application".
4
Under "Authorized JavaScript origins" add your GitHub Pages URL (e.g. https://yourusername.github.io). Also add http://localhost for testing.
5
Copy the Client ID and paste it above, then tap Connect.
6
To set up GitHub Pages: create a free account at github.com, make a new repository, upload this file as index.html, then go to Settings → Pages → Deploy from main branch.
