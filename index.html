One-time setup guide
1
Go to console.cloud.google.com and open or create a project.
2
Search "Google Drive API" and enable it.
3
Go to Google Auth Platform → Clients → Create OAuth Client ID → Web application.
4
Under Authorized JavaScript origins add https://millerwilbourn.github.io and http://localhost.
5
Copy the Client ID, paste above, and tap Connect.
